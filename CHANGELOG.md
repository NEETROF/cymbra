# Changelog

High-level, cross-cutting overview of the Cymbra monorepo.

Per-artifact changelogs are generated by **release-please** (e.g.
`apps/music/CHANGELOG.md`), and each release is published under **GitHub Releases**
with prefixed tags (e.g. `music-vX.Y.Z`). This file is a human overview only.

The format is based on [Keep a Changelog](https://keepachangelog.com/);
versioning is [Semantic Versioning](https://semver.org/) **per artifact**.

### Added
- Monorepo skeleton: Cargo workspace (Rust, edition 2024) + Melos (Flutter/Dart).
- `apps/music`: interactive piano POC (Flutter + Rust via `flutter_rust_bridge`) —
  real USB-MIDI on macOS, Linux, Windows, Android and iOS, dual rendering
  (staff / Synthesia waterfall) and a basic Wait Mode.
- CI quality gates: `rust`, `flutter`, `frb-codegen`, `sonar`, `commitlint`.
- Contribution governance: PR & issue templates, `CODEOWNERS`, `SECURITY.md`,
  `SUPPORT.md`, `FUNDING.yml`.
- Apache-2.0 license + trademark policy (`TRADEMARKS.md`).
- Automated releases via release-please.
