# Support

Thanks for using Cymbra! Here's how to get help.

## Questions & ideas
Use **GitHub Discussions**: <https://github.com/NEETROF/cymbra/discussions>
for usage questions, ideas, and general discussion.

## Bugs & feature requests
Open an **issue** with the matching template:
<https://github.com/NEETROF/cymbra/issues/new/choose>

Please include your platform (macOS/Linux/Windows/Android/iOS), the component
(e.g. `apps/music`), and — for MIDI problems — your device and any logs tagged
`[cymbra-midi]` / `[cymbra-swift]`.

## Security
Do **not** open a public issue for vulnerabilities — see [SECURITY.md](SECURITY.md).

## Documentation
Start with the [README](README.md) (monorepo overview) and the app README under
`apps/music/`.
